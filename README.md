# Dramatic Windows

Create dramatic Windows message boxes.

**Dramatic Windows** is a simple tool that lets you build sequences of classic Windows dialog windows with different icons (info, warning, error, question). Perfect for lyric videos and dramatic storytelling.
---

### What is this?

**Dramatic Windows** is a tool that lets you create sequences of classic Windows message boxes with custom text.

---

### ⚙️ Features

- Add lines one by one
- Choose window type:
  - ℹ️ Info (bluy)
  - ⚠️ Warning (yellow)
  - ❌ Error (red)
  - ❓ Question
- Custom window title
- Russian / English interface
- Uses real Windows system message boxes

### Что это

**Dramatic Windows** — программа для создания последовательности системных окон Windows с текстом.

Ты добавляешь строки, выбираешь тип окна (инфо / предупреждение / ошибка / вопрос), нажимаешь запуск — и окна появляются по очереди.

---

### ⚙️ Возможности

- Добавление строк по одной
- Выбор типа окна:
  - ℹ️ Info (синий)
  - ⚠️ Warning (жёлтый)
  - ❌ Error (красный)
  - ❓ Question
- Настраиваемый заголовок окна
- Поддержка русского и английского интерфейса
- Используются настоящие системные окна Windows
